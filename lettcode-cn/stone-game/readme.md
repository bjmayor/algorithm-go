试下dp求解。
定义dp[i][j]为剩下的石头为第i堆石头到j堆石头 当前玩家能获取的最大分数差
状态转移为：
dp[i][j] = max(piles[i]-dp[i-1][j],piles[j]-dp[i][j-1])
其中piles[i]-dp[i-1][j] 表示当前玩家拿了i
其中piles[j]-dp[i][j-1] 表示当前玩家拿了j
