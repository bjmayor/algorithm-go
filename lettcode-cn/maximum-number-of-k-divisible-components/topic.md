题目：2872. 可以被 K 整除连通块的最大数目

给定一棵包含 n 个节点（编号 0 到 n-1）的树，一个长度为 n 的整数数组 value，表示每个节点的值，以及一个整数 k。可以通过删除若干条边将树分割成若干个连通块。如果某个连通块中所有节点的值之和能够被 k 整除，则称该连通块为 k-可整除的。要求在至多删除任意条边的情况下，得到 k-可整除连通块的最大数量。

输入说明（函数式输入）：
- n：节点数
- edges：大小为 n-1 的二维数组，每个元素为 [u, v] 表示无向边
- value：长度为 n 的整数数组
- k：正整数

输出：能得到的 k-可整除连通块的最大数量（整数）

示例 1：

输入：n = 5, edges = [[0,2],[1,2],[1,3],[2,4]], values = [1,8,1,4,4], k = 6

输出：2

解释：一种合法的划分是删除边 1-2 和 0-2，将树分割成两个 k-可整除连通块：

- 块A：节点 {1,3}，值和 = values[1] + values[3] = 8 + 4 = 12，能被 6 整除。
- 块B：节点 {0,2,4}，值和 = 1 + 1 + 4 = 6，能被 6 整除。

示例 2：

输入：n = 7, edges = [[0,1],[0,2],[1,3],[1,4],[2,5],[2,6]], values = [3,0,6,1,5,2,1], k = 3

输出：3

解释：删除边 0-2 与 0-1，可以得到 3 个 k-可整除连通块：

- 节点 {0}，和值=3
- 节点 {2,5,6}，和值=6+2+1=9
- 节点 {1,3,4}，和值=0+1+5=6

思路提示：对树做一次 DFS，计算每个子树节点值之和对 k 的取模；当某个子树的和对 k 取模为 0 时，可以把该子树与其父节点之间的边切断，从而形成一个 k-可整除的连通块，计数加一。这是自底向上的贪心策略：当子树已满足可整除时立即切断不会降低最终答案。
