3228. 将 1 移动到末尾的最大操作次数

## 题目描述

给你一个二进制字符串 s。

你可以对这个字符串执行 **任意次** 下述操作：

选择一个下标 i（i + 1 < s.length），该下标满足 s[i] == '1' 且 s[i + 1] == '0'。
交换 s[i] 和 s[i + 1]，将 '1' 向右移动一位。

例如，对于 s = "010010"，如果我们选择 i = 1，结果字符串将会是 s = "001010"。

返回你能执行的 **最大** 操作次数。

## 示例

**示例 1：**

输入：s = "1001101"
输出：4
解释：
可以执行以下操作：
- 选择 i = 0。结果字符串为 s = "0101101"。
- 选择 i = 4。结果字符串为 s = "0101011"。
- 选择 i = 3。结果字符串为 s = "0100111"。
- 选择 i = 2。结果字符串为 s = "0010111"。

**示例 2：**

输入：s = "00111"
输出：0

## 提示

- 1 <= s.length <= 10^5
- s[i] 为 '0' 或 '1'
