3531. 统计被覆盖的建筑

## 题目描述

给定一个正整数 `n`，表示一个 `n * n` 的城市，同时给出一个二维数组 `buildings`，其中 `buildings[index] = [x, y]`，表示在坐标 `(x, y)` 有一个唯一的建筑。

若某个建筑在上、下、左、右四个方向中每个方向都至少存在一个建筑，则称这个建筑被覆盖。

问有多少建筑被覆盖。

## 约束条件

- `1 <= n <= 10^5`
- `1 <= buildings.length <= 10^5`
- `0 <= x, y < n`
- `buildings` 中的每个坐标 `(x, y)` 都是唯一的

## 解题思路

1. 使用两个哈希表分别统计每行每列的建筑坐标范围：
   - `row[x]` 存储第 `x` 行所有建筑的最小和最大 `y` 坐标
   - `col[y]` 存储第 `y` 列所有建筑的最小和最大 `x` 坐标

2. 对每个建筑 `(x, y)` 进行检查：
   - 如果 `x` 在 `col[y][0]` 和 `col[y][1]` 之间（包含边界）
   - 并且 `y` 在 `row[x][0]` 和 `row[x][1]` 之间（包含边界）
   - 说明该建筑被覆盖

3. 统计被覆盖的建筑数量
