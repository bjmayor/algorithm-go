四平方和定理 （英语：Lagrange's four-square theorem） 说明每个正整数均可表示为4个整数的平方和。它是费马多边形数定理和华林问题的特例。

欧拉发现的恒等式
(a^2 + b^2 + c^2 + d^2) * (x^2 + y^2 + z^2+w^2) = (ax+by+cz+dw)^2 + (ay-bx+cw-dz)^2 + (az-bw-cx+dy)^2 + (aw + bz -cy -dx)^2
