2141. 同时运行 N 台电脑的最长时间

题目描述

给你一个整数 n ，表示有 n 台电脑，以及一个下标从 0 开始的整数数组 batteries ，其中 batteries[i] 表示第 i 块电池可以提供的运行小时数。你可以在任意时刻将电池插入任意一台电脑。

每次一块电池只能插入并供一台电脑使用，但你可以随时替换电池（例如将电池从一台电脑上拔下来换到另一台电脑上）。当一块电池电量耗尽时它就不能再使用。

请你返回在最佳电池分配和替换策略下，使得所有 n 台电脑同时运行的最长小时数。

示例

示例 1：
输入：n = 2, batteries = [3,3,3]
输出：4
解释：可以将两块电池（容量 3 和 3）分别插入两台电脑，第三块电池可用于替换任意一台已耗尽电量的电脑，从而使得两台电脑同时运行 4 小时。

示例 2：
输入：n = 2, batteries = [1,1,1,1]
输出：2
解释：四块电池各自可提供 1 小时，总计 4 小时，平均到两台电脑上就是每台 2 小时。

示例 3：
输入：n = 1, batteries = [10]
输出：10

提示与约束

- 1 <= n <= 10^5
- n <= batteries.length <= 10^5
- 1 <= batteries[i] <= 10^9
